# 方向 C · B2：行业中性分解——纯风格 α + 行业轮动 β（2026-07-08 落地）

> 设计依据：roadmap 设计稿 §2.4 v2（行业中性版）。前置：B1 复现闸门通过
> （`2026-07-08-b1-style-basket-replication.md`，信号级 ρ 0.88-0.91）。
> 核心命题：指数对信号 = 纯风格轮动 + 行业轮动 + 编制怪癖 的混合体（§2.3 透视发现
> 行业主动暴露 45-73%）——B2 把前两个分量拆开、各自测预测力，**分解本身产生两个新信号**。

## 1. 方法

- **选样改造（唯一差异点）**：v2 = 每 CITIC 一级行业内 Top/Bottom 30%（`select_baskets_industry_neutral`）。
  两腿从每行业取相同只数且桶内等权 → **行业暴露=0 由构造保证**；打分完全复用 v1 的
  scores 缓存 → v1−v2 之差干净地等于纯行业配置分量（**rotation := v1.spread − v2.spread**）。
- **行业 PIT**：2021-01 起 67 期月度快照 asof（effective_date）；2021 前用每股最早标签
  静态外推（`extend_earliest_to_past`，设计稿定调的近似）；无标签票 623 只（多为 2021 前
  退市）与 <5 只小行业并入 pooled 桶同规则选样。
- TDD：5 新测试（行业内选样/并桶/单行业退化=v1/NaN 剔除/静态外推），全套 **131 过**。
- 构建：五套 `spread_<U>_neutral.csv` 各 3,178 天（篮子 ~353/腿 @U2，v1 为 366）。

## 2. ★ 结果（决策级）

### 2.1 方差分解——「混合体」定量坐实

U2 口径：**v1 混合价差年化波动 13.3% ≈ v2 纯风格 7.7% + rotation 6.8%**（corr(v2,rot)=0.65，
两分量共振非正交——市场级风格共振的另一面）。blend 指数对日收益对两分量回归 **R²=0.77**
（单独：纯风格 0.64 / 行业轮动 0.63）；1000 对 R²=0.76 最高。
→ 设计稿 §2.3「指数对=混合体」从透视推断升级为时序定量：**约各半**。

### 2.2 预测 IC（月频非重叠 n=149，20d40z+sm5 信号化，目标=指数对价差）

| 信号 | blend 月频 Spearman | 500对 | 1000对 |
|---|---|---|---|
| v1 混合（=B1 自建） | 0.126 | 0.073 | 0.092 |
| **v2 纯风格** | **0.179** | **0.122** | **0.140** |
| rotation 行业轮动 | 0.050 | 0.005 | 0.020 |

- **纯风格 > 混合，全 6 组目标（2 universe × 3 target）方向一致**，相对提升 ~40%；
  分窗（2014-19 / 2020-26）排序均成立 → 设计稿判据「纯风格 IC 显著优于混合体」**命中**；
- **rotation 月频无独立预测力**（≈0.05），仅日频短窗 d5/d10 ~0.10 有信息 → 行业轮动
  的可预测性在短窗、月频衰减——不入月频择时池，短窗语义留档（与 Phase3/4 空头短窗
  结论互为参照，但语义不同：这是行业配置轴不是风格短腿）；
- **Caveat（必须随行）**：IC 绝对水平集中于 2020+（2014-19 两信号皆 ≈0：v1 −0.008 /
  v2 +0.026；2020-26：v1 0.181 / v2 0.256）。风格动量族可预测性 regime 依赖，与三线
  基线分段结构一致——不是自建缺陷，是信号族共性；
- 方法学注意：信号参数 20d40z+sm5 沿用 equal_weight 生产默认（step-2 已在指数对上
  确认的甜点），非本轮拟合；但形态同族 → 参数先验含全样本信息的成分，风险低非零。

## 4. ★ 切主评估（2026-07-08 同日，用户选做）——净值层判据未过，**不切主**

三方同秤（`backtest/pure_style_eval.py`：baseline 全套口径 + 公共窗对齐 n=3,028，
bootstrap 500）。blend·full：

| 信号 | full Sharpe | long Sharpe | long MaxDD | 换手 | p |
|---|---|---|---|---|---|
| **equal_weight（生产基线）** | **1.39** | **1.80** | −16.7% | 21.1 | .004 |
| self_mixed_U2（自建混合） | 1.01 | 1.52 | −15.7% | 19.3 | .008 |
| pure_style_U2（纯风格） | 0.98 | 1.46 | −19.7% | 22.5 | .008 |

- **两步梯级都输**：落差主要来自「自建 vs 指数对」（1.01 vs 1.39），行业中性一步在
  净值层中性（0.98≈1.01，IC 提升未转化）；三信号 p 均 <0.01（都显著优于随机，输的是相对强度）；
- **IC↑ 而净值↓ 的解释（方法论发现，决策级）**：IC 的预测目标是**指数对价差**（风格轴
  自身），净值的交易标的是**宽基期货**（风格+行业+β 混合体，方差分解已证约各半）——
  纯风格是更好的"风格测量仪"，但宽基择时恰恰需要行业轮动成分；equal_weight 指数对
  信号与标的同生态（市值加权编制），混合体对混合体。换手相近（21 vs 22.5），非成本
  问题，是信号-标的匹配度问题；
- 信号相关 ew/mixed/pure = 0.906/0.877，同向日 85.5% → 高度同族，集成增量有限；
- **∴ equal_weight 保持生产主信号**（经受住自建挑战者同秤检验，本身即价值）；
  纯风格信号定位改为**风格测量仪**（更干净的风格轮动读数，研究/仪表盘用途），
  非宽基择时器。与 Phase3/4 证伪空头引擎同风格：负结果也是决策级产出。
