# 动量变换器扫描:成长/价值配对的判强尺子替换评估(2026-07-10)

> 用户开题:分组不动(仍是成长/价值四对),把配对判强弱的度量从"区间相对
> 收益"换成"相对动量"。brainstorm 三段(家族/机器裁决/交付)已逐段确认。
> 这**不是第九根信息轴**——同一信息源换变换器,裁决框架用 walk-forward
> 扫描 + 同秤头对头(step-2/切主评估同款),不用八轴三关闸门(同源变体
> 几乎必死在关 1b 偏 IC,那是未测先判,不适合替换命题)。

## 1. 命题

现任:`_compute_pair_signal` = 日收益差 → lookback 滚动复利(20d)→
z(40) → 四对等权 → 平滑(5) = 生产 equal_weight(long-flat Sharpe 1.42)。
替换候选:每腿算动量、配对取差,z→等权→平滑下游完全一致。问:**动量
变换器能否在同秤下打赢现任收益变换器?**

## 2. 因子家族(三族 14 形态)

| 族 | 定义(每腿) | 网格 |
|---|---|---|
| M1 经典跳月 | `P(t−skip)/P(t−L−skip) − 1` | L∈{60,120,250} × skip∈{0,20} |
| M2 趋势斜率 | log 价格对时间滚动 OLS 斜率 | L∈{20,60,120,250} |
| M3 风险调整 | 区间收益 / 同窗日波动 | L∈{20,60,120,250} |

下游:z_window∈{40,120,250} × smoothing∈{0,5} → **84 组合**,scan_grid
三窗(train 14-20 / val 21-23 / holdout 24-26,blend 口径 + carry,
mode=discrete,3bp)。

**先验登记(诚实口径)**:step-2 已扫过收益差 L∈{40,60} 且全退化 →
"纯拉长形成期(skip=0)"大概率已死;本轮真正的活假设是 **skip 跳月**
(剔近月反转污染)与**非端点度量**(斜率路径加权 / 波动归一)。
锚点:M1(L=20, skip=0) ≈ 现任,作恒等性 sanity 不进裁决。

## 3. 机器

新模块 `backtest/momentum_scan.py`,**不碰 `signals/` 生产代码**(零字节
回归风险),四层:

1. 动量纯函数:`momentum_classic(price, L, skip)` /
   `momentum_slope(log_price, L)`(向量化滚动 OLS,拒绝逐窗 apply)/
   `momentum_voladj(price, L)`;
2. 配对装配 `momentum_pair_factor(prices, ...)`:每腿动量 → 配对差 →
   z(z_window) → 四对等权 → 平滑(下游与生产逐步对齐,用生产函数对拍);
3. `momentum_factor_fn(family)` 接入现成 `scan_grid`(PG 8 列一次加载,
   equal_weight_factor_fn 同款);
4. 同秤头对头:每族高原代表上 baseline 全套口径(blend·full+分段、
   long-flat+对称、bootstrap 500)与现任 20d40z+sm5 并排。

产出:`backtest/output/scan_momentum.csv` + `momentum_head2head.csv`
(胜负皆入库,证伪也留证据)。

## 4. 裁决标准(预登记,防事后挑数)

**换尺子门槛,三条全中才谈替换**:

1. 三窗全正且是高原(worst-window ≥ 现任同窗);
2. holdout Sharpe 明确高于现任;
3. 全窗 long-flat Sharpe ≥ 1.42 且 MaxDD 不劣。

**平局/局部略优 → 不换**:同源变体预期相关 0.8+,无全面优势时替换只有
回归成本没有收益,结论记"收益变换器第三次确认"。corr(变体,现任)与
置换 p 只作诊断列不作闸门。反过拟合纪律沿用 step-2:拒绝 train 负、
holdout 高的尖峰;高原比较看三窗最差值。

## 5. 测试锚(TDD 先行)

1. 恒等锚:M1(L=20,skip=0) 因子与现任 factor_value 相关 >0.99;
2. slope 向量化 ≡ naive 逐窗 OLS(数值恒等);
3. voladj 小样本手算锚;
4. 装配层下游与生产 `calculate_contrast_equal_weight_signal` 同输入对拍;
5. skip 对齐语义钉死(`P(t−skip)` off-by-one);
6. 短历史/NaN 守卫(min_periods 语义)。

## 7. ★ 结果(2026-07-10 同日落地)——按预登记标准**不换**,但有星号

bootstrap 500,产出 `scan_momentum.csv` + `momentum_head2head{,_diag}.csv`。

**同秤头对头(blend·full,θ=0 各方一致;本 harness 现任 long-flat=1.78,
与 Phase3 headline 1.42 的差异是 θ/窗口 harness 不同,判据③按同秤执行)**:

**裁决(预登记三条)**:①三窗全正且 worst 1.38 > 1.27 ✓;②holdout
**1.69 vs 1.63〜1.69 = 平手级**,非"明确高于" ✗;③同秤 long-flat 全口径
不劣、MaxDD 更浅 ✓。**②未过 → 按预登记规则不换**——增益集中在
train/val(2014-23),对最近 regime 无 OOS 增量证据,corr 0.945 同族,
替换只有回归成本。

**假设逐条关帐**:skip 跳月**全灭**(无一进前 20);长形成期第三次
证伪(classic 全族远逊,与 step-2 收益差 L40/60 退化一致)——**"动量"
本义(长形成期/剔近月)在这个配对上彻底死了**;voladj 波动归一不及现任。

**★ 星号(净发现)**:活下来的不是动量,是**同频率的度量稳健化**——
slope L20(20 日对数价格 OLS 斜率,路径加权)在测到的全部 10 个格子里
不劣于端点收益,train/val 明确更稳(+0.11〜+0.14),MaxDD/换手双降。
**登记为生产信号第一替补**:若未来需要 regime 稳健性或重估因子秩序,
slope20 是首个应试变体(机器现成:`momentum_factor_fn` 一行可复算)。
收益变换器地位=第三次确认(切主、八轴、本轮)。

### 7.1 短窗补测(同日,用户追加"比如 5 日动量")——✗ 全灭,载频结论定型

网格扩 L∈{5,10} 三族(短窗只 skip=0,skip 已全灭且 skip≥L 无语义),
20 形态 × 120 组重扫(测试同步 84→120,全套 202 过)。**36 个短窗组合
全部远低于现任(worst 1.27)与 slope20(1.38)**:最好的是
voladj5/zw120/sm5 = worst 0.73;**用户假设的主角 slope 短窗恰恰最差**
(slope10 最好 0.47、slope5 最好 0.08,多组为负)——5 个点拟合直线是
噪声上再估噪声。与 step-2 端点收益 lb5/10(worst ≤0.40)互相印证。

**∴ 载频结论定型**:成长/价值相对强弱的信息载频就在 **20 日附近**——
5/10 日里没有信号可救(度量再稳也无中生有),60 日以上信息已死
(三族三次证伪)。度量稳健化(slope)只能在有信息的频率上放大信息,
不能改变信息在哪。**§7 裁决不变**(每族高原代表无一被短窗撼动,
head2head 无需重跑):不换,slope20 仍是第一替补。

### 7.2 短 z 窗补测(同日,用户追问"5/20 是否试过")——✗ 精确点全负

**5/20 确实两张网格都漏了**:step-2 的 z 网格跟随 lookback
(lb5 → zw{10,15,25,250},夹着 20 但无精确点),本轮动量网格 zw 固定
{40,120,250}。补针对性扫描:三族 × L{5,10} × zw{10,15,20,25,30} ×
sm{0,5} = 60 组 → `scan_momentum_shortz.csv`(ad-hoc,不进默认网格)。

**目标点 5/20 六组全负**(worst −0.47〜−0.96;classic/slope/voladj
无一幸免);60 组邻域最好 = classic10/zw30/sm5 的 worst **0.28**,与
step-2 端点 lb10/zw30/sm5=0.40 互证——短窗死透,与 z 窗无关、与度量
无关。step-2 旧证据同款:lb5 配 zw{10,15,25} 本就全负(−0.54〜−1.47)。

**⚠️ 副产品(生产口径警示)**:classic 5/20/sm0 ≈ **equal_weight 生产
变体 B(5d20z)本尊**(恒等锚同构)——即变体 B 在修秤口径(blend·
carry-aware·三窗)下 train −0.34 / val −0.67 / holdout +0.20,
**不具生产价值**。变体 B 是 2026-06 修秤之前的产物,从未过过这杆秤;
20d40z(变体 A)地位不受影响。是否弃用/降级变体 B 输出=用户决策
(committed 输出线,弃用破字节回归习惯,建议至少在 README 标注)。

### 7.3 国信研报三候选补测(同日,用户确认)——✗ 三族全灭,现任第五次确认

侦察《国信金工·动量类因子全解析》(2021-12,截面选股口径)后提取三个
close-only 可迁移新度量,以**配对级形式**(作用在相对序列,非每腿差)
